# Module 1 - 任务列表

<Steps>
### 第一步：编写生日贺卡（Birthday Card）

- **正确使用 HTML 和 CSS（10 分）**  
  - 你需要在生日贺卡中正确使用 HTML 和 CSS。确保使用 HTML 进行“内容”编写，使用 CSS 进行“外观”设置。不要编写内联样式（inline styles），使用独立的样式表并将其链接到你的 HTML 文件。

- **至少使用 5 种不同的 CSS 属性（10 分）**  
  - 在你的生日贺卡中尝试至少 5 种不同的 CSS 属性。

- **生日贺卡和搜索门户页面通过 W3C 的 HTML 5 和 CSS 验证（10 分）**  
  - 使用 [HTML 5 验证工具](https://validator.w3.org/#validate_by_input) 确保你的生日贺卡和 [CSS 验证工具](https://jigsaw.w3.org/css-validator/#validate_by_input) 确保你的 DuckDuckGo 搜索门户页面通过 W3C 的验证。

<Callout>
  一定要记得把自己代码粘贴进去检查哦！很多人都在这里扣分.
</Callout>

### 第二步：创建 DuckDuckGo 搜索（Search Portal）

- **DuckDuckGo 搜索门户功能正常（9 分）**  
  - 创建一个功能正常的 [DuckDuckGo](https://duckduckgo.com/) 搜索功能，界面跳转就行。

- **搜索门户页面包含课程的复议政策文本（1 分）**  
  - 确保在你的搜索门户页面中包含本课程的 [Regrade Policy](https://www.arl.wustl.edu/~todd/cse330/info.html) 。

<Callout>
  记得确保把policy复制到你的网页里.
</Callout>
